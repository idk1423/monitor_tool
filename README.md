# Network Tool

Консольная утилита для мониторинга сетевых подключений к популярным сервисам (Roblox, Discord, Steam, YouTube и др.)

## Установка и запуск

1. Скачайте [WinDivert](https://reqrypt.org/windivert.html) (64-бит) и положите WinDivert.dll рядом с network_tool.exe.
2. Скачайте network_tool.exe из этого репозитория.
3. Запускайте от имени администратора.
4. Чтобы запустить монитор, дважды кликните по run_monitor.bat
